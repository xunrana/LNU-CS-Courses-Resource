## 📚 核心专业大课真实资料盘点与备考攻略

### 1. 🟢 计算机组成原理 & 实验（3 + 1 学分）
本课程带你深入研究计算机底层硬件是如何通过电平控制、时序发生和数据通路来实现机器指令的取指与执行。
* **📂 理论课文件夹真实资料盘点**：
  * **权威教案与大纲**：
    * `张大波老师教学日历.doc` 与 `计组考核大纲.docx`：明确每节课内容与结课要求。
    * 课后习题标准答案 PDF（第一、六、七章作业）：重点覆盖了数据的机器数表示、运算器设计、微程序执行及指令系统分析。
  * **珍贵备考题库**：
    * 计组期末题库整理（**12MB 级超大 PDF/Word 文件**）：包含历年名校及辽大计组真题、易错题整理及解题过程。
    * **冯东晖老师 13级至18级 历年期末真题及标准答案**：包括A/B卷与答案详析，考前一定要独立模考！
* **📂 实验课文件夹真实资料盘点（TEC-8硬件开发与操作）**：
  * **实操讲解视频包**：
    * 包含 `0 实验系统介绍.mp4`、`1 寄存器实验.mp4`、`2 脱机运算器实验.mp4`、`实验3 双端口存储器.mp4`、`实验4 微程序 1.mp4`、`7 中断原理实验.mp4` 等多拍高清演示录像，手把手演示电平开关怎么拨、QD按钮怎么按。
  * **实验指导讲义**：包含手写标注版的 `TEC 8指导书新 stu.pdf` 与 `TEC 8 实验箱讲义v18.08版指导书`。
  * **六大经典硬件实验报告**：
    * **实验一：寄存器与运算器实验**。熟悉通用寄存器堆 RF 内部的 R0-R3 写入机制（使用 DRW、RD0/RD1 译码出控制信号）；验证 74LS181 构成的 8 位运算器在算术（加/减）与逻辑（与/或/非）运算下的进位 C 和零标志 Z。
    * **实验二：双端口存储器实验**。验证静态存储器 IDT7132 在左端口读写（MEMW、MBUS 信号控制）与右端口只读（由 PC 指向）的并发工作机制。
    * **实验三：数据通路实验**。将通用寄存器堆、双端口存储器与 74LS181 ALU 互连，测试多模块间的数据流向和总线 DBUS 的读写控制。
    * **实验四：微程序控制器实验**。理解时序发生器和微指令格式，设计 40 位字长（控制字段 29 位，后继微地址 11 位）的微程序流程图，实现微地址的条件转移控制。
    * **实验五：CPU组成与机器指令的执行**。模型机的核心实验，通过指令译码与硬件微指令配合，完成算术逻辑指令的执行通路。
    * **实验六：中断原理实验**。实现硬件中断请求的触发、中断响应、中断服务程序执行及中断返回（RETI）的全周期波形记录。
* **💡 备考突破秘籍**：
  1. **数据的机器数表示与计算（大题基础）**：熟练计算原码、反码、补码和移码的表示，以及补码加减法运算中的溢出判定方法（单符号位双符号位变形补码判定法）。
  2. **指令格式与寻址方式（高分计算题）**：根据给定的指令字长和寻址范围设计单字长/双字长指令格式。熟练推导直接寻址、间接寻址、寄存器寻址、变址寻址和相对寻址下有效地址（$EA$）的计算公式。
  3. **微程序控制器（核心难点）**：
     * 掌握微程序控制器的工作原理；
     * 根据机器指令功能，在微指令表中写出每一拍时序（T1, T2, T3）下需要开通的控制信号（如 `LPAR=1`, `DRW=1`, `MEMW=1` 等）；
     * 区分微程序控制与硬连线控制的优缺点。

### 2. 🟢 算法设计与分析 & 实验（3 + 0.5 学分）
本课程是衡量编程与数学逻辑的硬指标，主要讲授各类经典算法设计策略及其时间与空间复杂度的严谨推导。
* **📂 理论课文件夹真实资料盘点**：
  * **教学大纲与日历**：
    * `徐红艳老师教学日历.doc` 与 `理论课大纲.doc`。
    * [算法设计与分析——各章内容总结.docx](3%20算法设计与分析/算法设计与分析——各章内容总结.docx)：期末划重点的大纲总结！包含了算法时间复杂度（主定理三大 Case，递推方程求解）、分治（芯片测试、归并/快排）、动态规划（0-1背包、LCS）、回溯和分支限界的必考概念。
  * **高价值整理包**：
    * 包含 `算法习题解答.pdf` 和 `算法教材.pdf`。
    * **徐红艳老师 21 年算法考试真题** 卷、18 届算法真题卷，以及李响老师硬件方向多套 A/B 卷真题和答案。
  * **社会场景“真实问题”表**：
    * 选题1：基于**遗传算法**的现代物流配送路径优化问题（TSP的变体解决）。
    * 选题2：多任务环境中基于**动态优先级**的计算资源高效分配与调度算法。
* **📂 实验课文件夹真实资料盘点（已重构代码分类）**：
  * **6次纯C++上机实验**：包含标准源码（`.cpp`）、编译后的 `.exe` 验证程序、PPT 讲解课件和**讲解录屏视频**（`1.mkv` 到 `6.mkv`，总大小近 600MB）：
    * **实验一：分治法**。求解二叉链表存储下二叉树从根到叶最大路径和；分治策略求有序递增序列的众数；分治法求解逆序对数。
    * **实验二：动态规划**。实现 0-1 背包问题、最长公共子序列（LCS）、以及矩阵连乘最少乘法次数计算。
    * **实验三：贪心算法**。实现背包问题（分数背包）、单会场活动安排调度问题。
    * **实验四：多种方法求解旅行商问题 (TSP)**。使用分治与近似贪心策略对 TSP 问题进行实现和对比。
    * **实验五：回溯法**。经典 N-皇后问题求解、0-1 背包的回溯搜索、旅行售货员问题的回溯。
    * **实验六：分支限界法**。FIFO 与优先队列式分支限界法求解单源最短路径及 0-1 背包。
* **💡 备考突破秘籍**：
  1. **递归方程时间复杂度计算（必考填空选择）**：
     * **迭代法**（逐步展开递推项）。
     * **主定理（Master Theorem）**：熟记 $T(n) = aT(n/b) + f(n)$ 对应的三种 Case。如 $T(n) = 2T(n/2) + \Theta(n)$ 属于 Case 2，时间复杂度为 $\Theta(n \log n)$。
  2. **分治法与动态规划的区别**：分治法适合子问题相互独立的情况；动态规划适合子问题重叠且具有最优子结构的情况，需要写出**状态转移方程**（如最长公共子序列的 $C[i][j]$，0-1 背包的 $V[i][j]$）。
  3. **贪心算法的正确性证明**：贪心选择性质与最优子结构性质。常考哈夫曼编码的构造、活动安排问题的按结束时间递增排序策略。
  4. **回溯法与分支限界法的搜索机制（大题核心）**：
     * **回溯法**：深度优先（DFS）搜索解空间树，通过剪枝函数（约束函数/限界函数）剪去无效分枝。必考子集树（如 0-1背包）与排列树（如旅行商问题）的搜索步骤。
     * **分支限界法**：广度优先（BFS）或最小耗费/最大键值优先搜索。使用队列或堆结构保存活结点，设计限界值计算公式以尽快淘汰次优分枝。

### 3. 🟢 面向对象程序设计（2 学分）
以 Java 为核心的高级面向对象语言课，重在理解封装、继承、多态，并具备编写复杂系统级项目的能力。
* **📂 理论课与大作业真实资料盘点**：
  * **JAVA大作业一：《面向电影编程（MOVIES-OOP）》**：
    * **系统设计**：基于 Java SE 17 + MySQL + Navicat 开发的电影推荐与管理系统。采用标准的多层架构开发：
      * `dao` (数据访问层)：实现 `MovieDao`、`ActorDao`、`DirectorDao`、`UserDao`、`UserFavoriteDao` 等对数据库的 CRUD 交互。
      * `model` (实体层)：定义电影（Movie）、导演（Director）、演员（Actor）、用户（User）的对象属性和封装。
      * `view` (展示层)：基于 Java Swing 窗口组件开发的 `LoginFrame`、`RegisterFrame`、`AdminMainFrame`（管理员主页面）与 `UserMainFrame`（用户检索收藏页面）。
    * **文档配置**：包含近 1MB 的完整满分 `开发文档 (2).docx/.pdf` 模板，以及建表 SQL 语句。
  * **JAVA大作业二：《逻辑代数的形式转化系统》**：
    * **功能**：输入逻辑代数表达式（如 `AB+AC`），系统自动进行词法合法性检查，并能够快速输出其反演式、对偶式、生成真值表，以及**基于 Quine-McCluskey 算法求出最简与或式**、标准与或/或与式等。
    * **核心类**：[GUI.java](2%20面向对象程序设计/JAVA大作业/课程资料目录/面向对象%28报告+项目%29/src/GUI.java)、[Simplest.java](2%20面向对象程序设计/JAVA大作业/课程资料目录/面向对象%28报告+项目%29/src/Simplest.java)、`Tools.java`（含大量的位运算和字符串操作辅助函数）。
  * **平时作业**：
    * 第一次雨课堂作业：设计银行账户管理（`Account` 类）、支票账户派生子类（`Checking` 类）及二维空间定位逻辑（`Location` 类）。
    * 第二次雨课堂作业：实现可比较圆对象 `ComparableCircle` 及可着色多边形接口控制（`Colorable`、`GeometricObject`、`Square`）。
* **💡 备考突破秘籍**：
  1. **面向对象三大特性**：深刻理解封装的作用；子类继承父类的初始化顺序（先执行父类构造函数）；多态的实现机制（动态绑定与方法重写 `Override`）。
  2. **抽象类与接口的区别**：接口（`interface`）只能包含常量和抽象方法（Java 8 后可有 default 方法），不能实例化，支持多继承；抽象类（`abstract class`）可以包含普通成员变量和具体方法，只支持单继承。
  3. **Java 异常处理与集合框架**：掌握 `try-catch-finally` 的执行流；熟悉常用容器类（`ArrayList`, `LinkedList`, `HashMap`, `HashSet`）的特点与选择场景。

### 4. 🟢 数字逻辑 & 实验（3 + 0.5 学分）
这是计算机硬件方向的敲门砖，研究如何使用基本的门电路和逻辑器件来处理二进制的电平世界。
* **📂 理论课文件夹真实资料盘点**：
  * **教学课件与材料**：包含数字逻辑完整官方课件 PPT（涵盖编码、组合电路、触发器、时序电路、PLD 等）、配套课后习题详细解答 PDF。
  * **大题与真题大礼包**：
    * `数字逻辑练习题.pdf` 与 `数字逻辑整理.docx`。
    * **薛军/刘建老师 14 套历年期末真题与标准答案**（大一统经典考卷，时间横跨 09级至16级）：考前必做！考试大题和填空的题型和考点在这 14 套试卷中几乎达到了 95% 以上的重复率！
* **📂 实验课文件夹真实资料盘点**：
  * **电路讲义**：`数字电子技术实验讲义23.pdf`，详细指导芯片管脚连接。
  * **实验大纲说明**：`数字逻辑实验实验教学大纲.doc`。
  * **实验报告整理**：包含多个版本的 `数字逻辑 .docx` 实验报告，图文并茂，详细展示了芯片在面包板上的接线方法及实验输出记录。
* **💡 备考突破秘籍**：
  1. **逻辑代数基础与卡诺图化简（必考大题，送分点）**：掌握逻辑代数的基本公式（特别是摩根定律/反演律），熟练在卡诺图（Karnaugh Map）中圈出最小项进行最简与或式的化简，注意不要漏掉约束项（非法输入状态）。
  2. **组合逻辑电路的分析与设计（15分大题）**：
     * **分析**：已知逻辑电路图 -> 写出各级逻辑表达式 -> 化简并列出真值表 -> 确定电路功能。
     * **设计**：根据需求列出真值表 -> 写出逻辑表达式 -> 卡诺图化简 -> 转换为指定门电路（如全与非门）并画出电路图。
     * 掌握中规模集成电路（译码器 74LS138、数据选择器 74LS151）实现任意逻辑函数的方法。
  3. **时序逻辑电路的分析与设计（20分压轴大题）**：
     * 掌握基本触发器（RS、JK、D、T触发器）的特性方程。
     * **时序电路分析步骤**：写出各触发器的激励方程和输出方程 -> 代入特性方程写出状态转移方程 -> 列出状态转移表 -> 画出状态转移图并分析电路是否具有自启动能力。
     * **时序电路设计**：根据状态转换图 -> 确定触发器个数及类型 -> 写出状态转移表与卡诺图化简 -> 得到激励方程 -> 画出逻辑电路图。

### 5. 🟡 工程数学（二） & 应用数理统计（3 + 2 学分）
这是大学阶段最后也是最重要的数学工具课，为后续机器学习、人工智能和数据科学打下数理底子。
* **📂 工程数学（二）真实资料盘点（复变函数与积分变换）**：
  * **电子教材与习题**：包含经典的复变函数教材习题解答 PDF、积分变换教材习题解答 PDF，以及《工程数学笔记.pdf》整理资料。
  * **名师期末真卷库**：收录了罗浩、宋琪、段苛苛、刘砚秋等数位骨干教师在不同学期的**共计 18 套期末 A/B 卷真题及详细标准答案**（计算机与通信系专用）。考前重点刷罗浩老师的试卷，题风严谨。
* **📂 应用数理统计 真实资料盘点**：
  * **备考复习大礼包**：
    * `01 复习（老师发的）`、`02 复习（整理的简单易背版）`、`03 复习（空白默写版）`：形成了完美的“看背-背记-空白默写自测”复习闭环！
    * [04 题目及答案整理.pdf](2%20应用数理统计/学长%20应用数理统计/04%20题目及答案整理.pdf)：高达 5.5MB，精选了数理统计所有章节的典型例题与详细解答。
    * **22年及23年春季学期期末考试真题与答案**，复习风向标。
* **💡 备考突破秘籍**：
  1. **工程数学（二）**：
     * **复变函数**：求解析函数的充要条件（柯西-黎曼方程 C-R 方程）；利用柯西积分公式及留数定理（Residue Theorem）计算闭合曲线上的复积分。
     * **积分变换**：求常用函数的傅里叶变换与逆变换；使用拉普拉斯变换求解线性常系数微分方程的初值问题。
     * **数理方程**：掌握分离变量法求解一维弦振动方程与热传导方程的边界值问题。
  2. **应用数理统计**：
     * **参数估计**：求未知参数的矩估计（用样本矩代替总体矩）与极大似然估计（MLE，构建似然函数并求导数极值点）。
     * **假设检验**：分清双侧检验与单侧检验下拒绝域的构造；熟练运用 U 检验（方差已知，求均值）、T 检验（方差未知，求均值）、卡方检验（方差检验）和 F 检验（两总体方差比检验）。
     * **回归分析**：求一元线性回归方程中最小二乘估计的斜率 $\hat{b}$ 与截距 $\hat{a}$，并进行回归显著性检验。

## 💡 学长寄语

1. **“计组和算法决定了专业课的深度”**：大二春季学期是课表最满的时期之一，但计组和算法两门硬课绝不能抱有侥幸心理。直接打开我们归档的 [寄存器实验](1%20计算机组成原理实验/xxxx/1/—第一次实验寄存器.docx) 和 [运算器实验报告](1%20计算机组成原理实验/xxxx/1/—第一次实验运算器.docx)，结合 TEC-8 视频理解总线竞争和微地址跳转的实质。理论考试上，多做冯东晖老师的往年真题；算法上，一定要自己在本机实现一下动态规划最长公共子序列（LCS）和分支限界大题，这是将来面试手撕代码的常客。
2. **“Java大作业是积累项目背景的最佳时机”**：Java 大作业的 `MoviesOOP`（电影推荐系统）和 `逻辑代数转化器`，包含了完整的 Swing 桌面 GUI 框架、MySQL 数据库读写和高内聚低耦合的分层面向对象封装。好好阅读大作业中的 [MoviesOOP 开发文档](2%20面向对象程序设计/JAVA大作业/xxxx/开发文档%20%282%29.docx)，把多级实体映射关系梳理明白，这会成为你简历上第一个亮眼的项目背景。
3. **“数学和物理不要拖后腿”**：罗浩老师的工程数学（二）给分非常严格，千万不要觉得上课听懂了就可以，期末大题如果不手练三四套题，很容易在柯西留数计算时卡壳；数理统计的备考建议直接使用学长整理的 `02 复习（整理的简单易背版）` 记住核心统计量，并在 `03 空白默写版` 上闭卷自测，这样考前能保证极大似然估计和区间检验稳稳拿分。
