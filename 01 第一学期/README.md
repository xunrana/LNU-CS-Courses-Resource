## 🎯 重点提醒：英语四六级报考规则

> [!IMPORTANT]
> **以下规则来自当届经验整理，可能随教务处和学院通知调整。**
> 在当届规则中，四级考试的报考资格与大学英语期末考试成绩挂钩。尽早通过四级，能为大二、大三专业课学习腾出时间。

### 📊 报考门槛与判定流程（当届经验）：
1. **第一次报考机会（大一下）**：
   *   **门槛**：大一上学期《大学英语（一）》期末成绩必须达到 **85 分及以上**。
   *   **说明**：达到该分数通常可在大一下学期直接报考四级，具体以当年报名通知为准。
2. **第二次报考机会（大二上）**：
   *   **门槛**：如果大一上没过 85 分，则需要**大一两个学期英语期末的平均分达到 85 分及以上**。
3. **后续报考机会（大二下及以后）**：
   *   无期末成绩分数门槛限制，均可自由报考。
4. **如果大一下第一次四级没过，何时能考第二次？**
   *   需要前两个学期期末平均过 **85 分**，才可以在大二上考第二次；否则，必须等到大二下才能报考。

> [!TIP]
> 建议大一上英语尽量争取 85 分以上；实际报名资格以当年教务处和学院通知为准。

## 📚 各课程真实资料盘点与深度备考指南

### 1. 🟡 高等数学A（一）（5 学分）
高数是大一学分最高、挂科率较高、绩点占比较重的科目。一元微积分是后续工科数学的重要基础。
*   **📂 文件夹内真实资料盘点**：
    *   `1.docx`、`2.docx`：高数上核心定理、重要公式与经典例题总结。
    *   `2015级高数（上）期末考试试题.pdf` 与 [参考答案.doc](5%20高等数学A（一）/15级高数（上）期末考试试题及参考答案.doc)
    *   `2016级高数（上）期末考试试题.pdf` 与 [参考答案.doc](5%20高等数学A（一）/16级高数（上）期末考试试题及参考答案.doc)
    *   `2017级高数（上）期末考试试题.pdf` 与 [参考答案.pdf](5%20高等数学A（一）/17级高数（上）期末考试试题及参考答案.pdf)
*   **💡 备考建议**：
    1. **多理解、狂刷课后习题**：同济版教材课本上的课后习题极度重要，期末考试有近 **30%-40%** 的大题是由课后原题直接改编而来！务必把每章的课后习题亲手写一遍。
    2. **吃透 3 套历年真题**：辽大的高数期末考卷题型有一定重复规律（极限洛必达法则计算题、一元微分应用、微分中值定理罗尔/拉格朗日证明大题、定积分与不定积分计算）。考前两周建议闭卷模拟这三套真题，吃透其参考答案的**步骤分扣减细则**。

### 2. 🟢 计算机导论实验 (0.5 学分) 与 计算机导论 (2 学分)
计算机专业的入门启蒙课。理论偏向科普，实验课侧重办公三件套的高级应用以及基础编程环境的熟悉。
*   **📂 理论课真实资料盘点**：
    *   `xxxx.pptx`：涵盖计算机软硬件、二进制数据表示、操作系统、计算机网络等全领域的**超大综合课件**。
    *   [考试练习.docx](2%20计算机导论/考试练习.docx)：汇总了期末理论机考选择与简答的**核心题库**，适合考前背记。
*   **📂 实验课真实资料盘点 (6次完整实验)**：
    *   `计算机导论实验教学大纲.docx`、`教学日历.doc`。
    *   [学长 & 实验二](0.5%20计算机导论实验/实验一%2024%2011.14) (Word部分)：包含多级列表、页眉页脚、EX2 (图文混排)、EX3 (复杂表格与结果PDF对照)、EX5 思考与实战 (海报样式、活动日程安排排版)。
    *   [实验三 & 实验四](0.5%20计算机导论实验/实验三%2024%2011.30) (Excel部分)：EX6 (员工学历与销售记账基础)、EX7 (汽车零配件数据排序与筛选)、EX8 (利用 VLOOKUP, IF 函数进行复杂计算与成绩单管理)、EX9 思考与实践 (利用第五、六次全国人口普查数据生成复杂数据透视表与统计图)。
    *   [实验五](0.5%20计算机导论实验/实验五%2024.12.12) (C/C++基础编程)：提供 C 和 C++ 两个版本的入门代码（1.cpp - 4.cpp，均含已编译好的 .exe），涉及基础控制台输入输出、循环与条件分支。
    *   [实验六](0.5%20计算机导论实验/实验六%2024%2012.19) (PPT制作)：包含 33MB 大型的 `实验六.pptx`，以《大数据领域技术标准、知识产权、产业政策和法律法规》为题，展示了专业的幻灯片设计、主题美化与动画排版。
*   **💡 备考建议**：
    1. **理论课多读PPT，死记练习题**：期末理论考试多为概念选择，考前把 `考试练习.docx` 里的基础简答题与常用进制换算概念背过，即可按要求完成。
    2. **实验课多动手**：实验分值较高。不要因为 Office 常见就掉以轻心。期末上机重点考查**Word长文档多级排版自动目录生成**与**Excel复杂VLOOKUP跨表函数及数据透视表生成**。考前建议使用仓库中的 EX8、EX9 原始数据包自己从头实操做一遍。
    3. **编程起点**：实验五是编程生涯的第一步，跑通 `1.cpp - 4.cpp`，为第二学期的《高级语言程序设计》提前感知编译器的世界。

### 3. ⚪ 大学英语（一）（4 学分）
通识主干课，期末成绩直接影响四级首考资格。
*   **📂 文件夹内真实资料盘点**：
    *   [大学英语（一）大纲文件.doc](4%20大学英语（一）/大学英语（一）%20教学大纲.doc)：包含考核标准和平时分组成部分的课程大纲。

    2. **真题题型对标四级**：期末考试的听力、仔细阅读、词汇匹配等均是标准的四级真题难度。建议平时复习时直接拿英语四级真题刷题，既可以冲刺期末 85+，也能为大一下四级首考做准备。

### 4. ⚪ 思想道德与法治（3 学分）
思政系列第一门，课本厚、考点碎、字数多，纯靠逻辑常识容易失分。
*   **📂 文件夹内真实资料盘点**：
    *   [《思想道德与法治》复习提纲.docx](3%20思想道德与法治/《思想道德与法治》复习提纲.docx)：提炼了全书要点，适合考前速记。
    *   [思修笔记电子版.docx](3%20思想道德与法治/思修笔记电子版.docx)：核心答题模板与重点常识整理。
*   **💡 备考建议**：
    1. **考前突击多背**：理科生不建议裸考。考前 1-2 周，认真打印本仓库内的 `复习提纲` 与 `电子版笔记`，每天抽出 1 小时速记核心道德观、法律红线和人生观概念。
    2. **主观大题答题格式**：期末大题切忌一段话写到底。一定要写得有条理（采用 `①` `②` `③` 序号分段标出），多写一些结合教材的术语，哪怕概念没有完全背准，条理清晰、字迹端正也能捞到绝大多数步骤分。

## 💡 学长寄语

1. **高数第一**：大一尽量避免挂科，尤其是 5 学分的高数，挂科不仅拉低绩点，还会严重影响后续《高数二》《算法设计》等的修读。
2. **英语争取85**：大一上英语期末很重要。建议以 85 分为目标，尽早满足当届四级报名条件，后续专业课阶段压力会小一些。
3. **实验课多动手**：导论实验课是你在大学敲代码、做规范文档的起点，敷衍实验作业，到了后面大二写长篇实验报告、做项目管理时会吃大亏。
4. **适应大学生活**：大一上课程并不算繁重，是养成独立自学习惯、打下专业底子的好时期。学习节奏从第一学期开始建立。
