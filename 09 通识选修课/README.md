本课程是电子信息与微电子学科核心的专业大课，逻辑严密，计算量繁琐。

* **📂 理论课真实资料盘点**：
  * **教学大纲与教学日历**：

    * [10073—01—电路分析基础 教学日历.doc](3%20电路分析基础/10073—01—电路分析基础%20教学日历.doc)：规划了各章节复习思考题题号。
  * **考前复习秘籍**：
    * [电路分析基础笔记.pdf](3%20电路分析基础/电路分析基础笔记.pdf)：高达 **60MB 的全书手写重难点笔记**，涵盖了电感电容复频域模型变换等手绘过程。
    * [电路分析基础习题.pdf](3%20电路分析基础/电路分析基础习题.pdf)：**80MB 级超大课后习题详细答案册**。
  * **真题演练卷**：
    * 包含了 [电路考试题.docx](3%20电路分析基础/电路考试题.docx)、[考试题选择题.pdf](3%20电路分析基础/考试题选择题.pdf) 以及 [考试题综合题.pdf](3%20电路分析基础/考试题综合题.pdf)，考前必做！

* **💡 期末必考大题型归纳**：
  1. **网孔电流法/结点电压法（含受控源）**：
     * **重点**：对受控源（如 $g_m U$ 或 $\beta I$）先当作独立源列回路 KVL 或结点 KCL 方程，随后补充一个受控源控制量与回路电流/结点电压的**辅助方程**进行联立消元。
  2. **戴维宁定理等效参数求解**：
     * **重点**：将待求二端口网络开路，求出开路电压 $U_{oc}$。在求解等效内阻 $R_{eq}$ 时，若网络内**不含受控源**，可直接将独立源清零（电压源短路，电流源开路）后利用串并联求解；若**含有受控源**，必须采用**外加独立源法**（在端口外加电压源 $U$ 求流入电流 $I$，则 $R_{eq} = U/I$）或**开路电压与短路电流比值法**（$R_{eq} = U_{oc} / I_{sc}$）。
  3. **一阶电路时域分析的“三要素法”**：
     * **核心公式**：
       $$f(t) = f(\infty) + [f(0_+) - f(\infty)] e^{-t/\tau}$$
     * **步骤**：
       1. 确定初始状态值 $f(0_+)$（利用换路定则，电容电压 $u_C(0_+) = u_C(0_-)$，电感电流 $i_L(0_+) = i_L(0_-)$）。
       2. 确定稳态值 $f(\infty)$（换路后 $t \to \infty$ 时，电容等效为开路，电感等效为短路）。
       3. 确定时间常数 $\tau$：$\tau = R_{eq}C$（电容电路）或 $\tau = L / R_{eq}$（电感电路），其中 $R_{eq}$ 是从储能元件两端看进去的戴维宁等效电阻。
  4. **线性动态电路的“复频域拉普拉斯分析法”（运算法）**：
     * **步骤**：把时域电路图转化为 $s$ 域运算电路。
     * 电阻 $R \implies R$。
     * 电容 $C \implies$ 复阻抗 $1/sC$ 并联一个初值电压源象函数 $C u_C(0_-)$。
     * 电感 $L \implies$ 复阻抗 $sL$ 串联一个初值电流源象函数 $L i_L(0_-)$。
     * 应用基尔霍夫定律的 $s$ 域象函数形式求解电压、电流象函数，最后通过**部分分式展开法**进行拉普拉斯反变换求出时域全响应。

## 💡 学长寄语

1. **“通识选修课是刷绩点的主力军”**：大学期间有些必修专业课难度极高，极易拉低平均学分绩点（GPA）。因此，在修读通识选修课时，务必挑选给分大方、多优秀、考核简单的科目（如《犯罪心理学》、《微电子技术》）。直接参考仓库内的论文模板与大作业解答，花最少的时间拿最高的绩点！
2. **“电类核心基础不可松懈”**：如果是计算机或微电子方向的同学，修读《电路分析基础》这类技术大课时切不可掉以轻心。多花时间推演网孔法、一阶时域三要素法，把仓库里的笔记和习题库刷熟，保证专业课不拖后腿！
3. **“重视学术伦理道德”**：在完成任何论文考查与未来的毕业设计时，务必遵守学术诚信规则。期末大论文一定要用查重工具过滤一下，避免大段抄袭，为之后的毕业答辩做好演练。
预祝大家在通识选修课上绩点起飞，门门优秀！
