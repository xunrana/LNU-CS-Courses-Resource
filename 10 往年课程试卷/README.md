# 📖 辽宁大学信息学院往年试卷库 · 索引手册

> 本目录（`10 往年课程试卷`）是辽宁大学信息学院往年期末试卷、答案及试卷分析报告的精整归档库。
> 本目录共收录 **57 门课程** 的 **320 份** Word 试卷文档。当前 README 已覆盖全部课程文件清单，其中前 39 门课程包含适用专业、覆盖年级、出题教师、文档类型等扩展详情；后 18 门课程先提供简明索引，便于快速检索。
> 试卷主体集中在 **12级～17级、2011–2018 学年**；少量文件名保留原始异常年份或跨年字段，检索时建议优先看课程名、教师名和文档类型。

## 📊 各专业试卷分布统计

| 专业 | 涉及课程数 | 核心课程举例 |
| :--- | :---: | :--- |
| **计科**（计算机科学与技术） | 28 门 | 数据结构、操作系统、编译原理、计算机系统结构、组成原理、人工智能 |
| **软件工程** | 18 门 | 面向对象程序设计、UML与软件设计模式、软件工程导论、数字逻辑、编译原理 |
| **信息管理**（信息管理与信息系统） | 16 门 | 管理信息系统、决策支持系统、电子商务、管理学、概率论与数理统计 |

| **通信工程** | 5 门 | 移动通信、扩频通信、现代通信技术、通信原理、工程数学 |
