## 📚 核心专业大课真实资料盘点与备考攻略

### 1. 🟢 数据结构 & 实验（3 + 1 学分）
作为计算机最核心的内功课程，《数据结构》不仅在期末考试中占分极大，更是后续算法课、考研408以及大厂面试必问的终极科目。
* **📂 理论课文件夹真实资料盘点**：
  * **教师教案课件**：
    * `数据结构 刘婷婷ppt`：包含 10 个最新版 PPT，重点讲解了树、图、查找、排序算法。
    * 经典分类 PPT（绪论、线性表、栈和队列、树和二叉树、图、查找、排序）：这套教案经过美化，动画演示清晰，是梳理概念的首选。
  * **核心习题与大纲**：
    * `教学大纲.doc` & `教学日历.doc`（王青松老师版本）：精确规定了学习进度与考试范围。
    * [最短路径的测试题.docx](3%20数据结构/最短路径的测试题.docx)：图论大题最强模板，详细展示了 Dijkstra 和 Floyd 算法每一步的 $S$ 集合变化与最短路径表格推导过程。
  * **考前整理包**：
    * `学长 数据结构` 文件夹：包含经典练习题、标准数据结构 C/C++ 代码、排序算法对比总结以及 8MB 的《数据结构.考研试题.pdf》，供课后拓宽思维。
* **📂 实验课文件夹真实资料盘点 (已分类并规范命名重构)**：
  * **[00实验一 · 顺序表](1%20数据结构实验/00实验一)**：实现顺序表的初始化、插入、删除操作；分析随机插入和删除的时间复杂度 $O(n)$。
  * **[01实验二 · 单链表](1%20数据结构实验/01实验二)**：包含非递减单链表插入新元素保持有序，以及如何**使用双指针设计删除带头结点链表中倒数第 $k$ 个结点的算法**。
  * **[02实验三 · 栈的应用](1%20数据结构实验/02实验三)**：设计顺序栈的基本操作，并利用栈实现**数制转换**（十进制正整数输入，十六进制数输出，核心是商和余数的进出栈逻辑）。
  * **[03实验四 · 二叉树遍历与操作](1%20数据结构实验/03实验四)**：递归建立二叉树（以 '#' 代表空结点），实现先序、中序、后序遍历，并包含**不使用全局变量、仅用递归函数求解二叉树叶子结点个数和度为 1 结点个数的进阶算法**。
  * **[04实验五 · 队列与二叉树宽度](1%20数据结构实验/04实验五)**：基于链队列实现二叉树的**层次遍历**、求二叉树的宽度（结点数最多那一层的结点数）、以及判断一棵二叉树是否为完全二叉树。
  * **[05实验六 · 图与邻接表](1%20数据结构实验/05实验六)**：利用**邻接表**建立有向图，实现深度优先搜索（DFS）遍历，求解各顶点的出度和入度，以及删除指定弧尾和弧头的边。
  * **[06实验七 · 查找表应用（图书系统）](1%20数据结构实验/06实验七)**：使用顺序查找表技术开发**图书信息管理系统**（包含 ISBN 号、书名、价格），实现图书录入、ISBN/书名多重查找、以及图书价格冒泡排序。
  * **[07实验八 · 单链表综合应用（单词系统）](1%20数据结构实验/07实验八)**：设计并维护以单词为结点的链表系统。包含无序单词链表字典顺序排序、两个有序单词链表合并、特定单词删除以及统计符合指定前缀（`Prefix`）的单词个数。
* **💡 备考突破秘籍**：
  1. **线性表、栈与队列**：重视循环队列的“队空”与“队满”判定条件计算、多栈共享空间的内存指针边界，以及利用栈进行中缀表达式向后缀表达式（逆波兰式）转换的推导。
  2. **二叉树三大推导（必考）**：二叉树的性质推导（如度为 0 和度为 2 结点关系：$n_0 = n_2 + 1$）、根据先序+中序（或后序+中序）构造出唯一二叉树并写出另一遍历序列、哈夫曼树（Huffman Tree）及编码设计（带权路径长度 $WPL$ 计算）。
  3. **图论三大核心算法（重难点）**：
     * **生成树**：Prim 算法（侧重顶点，适合密网）与 Kruskal 算法（侧重边，适合稀疏网）的区别与执行过程。
     * **最短路径**：Dijkstra 算法求单源最短路径、Floyd 算法求多源最短路径，考试大题步骤一定要规范（参考 `最短路径的测试题.docx` 中的表格表示法）。
     * **拓扑排序与关键路径**：求 AOV 网的拓扑序列；求 AOE 网的关键路径（需要算出每个事件的 $ve(i)$、$vl(i)$ 和每个活动的 $e(i)$、$l(i)$，时差为 0 的即为关键活动）。
  4. **查找与排序（计算题大户）**：
     * **查找**：折半查找的过程（构造二叉判定树、计算平均查找长度 $ASL$）；二叉排序树（BST）的插入和删除；哈希表构造（除留余数法）与冲突处理（开放定址法/链地址法）下 $ASL$ 的计算。
     * **排序**：快速排序的划分过程（双指针碰撞交换）、堆排序（建初始大顶堆、调整堆的过程）、归并排序与希尔排序的增量序列演变。

### 2. 🟢 数据库原理 & 实验（3 + 1 学分）
本课程全面讲解关系型数据库的设计、开发与管理，重在 SQL 语言在工程项目中的实操运用。
* **📂 理论课文件夹真实资料盘点**：
  * **官方 PPT23**：包含极其系统的教学教案课件，覆盖了从数据库概述、关系运算理论、结构化查询语言 SQL 到数据库安全性与设计规范的全部重难点。
  * **平时作业与练习**：
    * `SQL作业` 目录：包含真实平时 SQL 作业代码（11.10.txt等）。
    * `SQL.docx`：**数据库理论精髓练习库**！包含对 Anne Smith 职工信息的查询、多表关联查询、以及如何对总成本超过100万项目的员工进行多级嵌套查询的设计。特别是强调了“同名判定”中 `e.empno` 唯一标识符的必要性。
  * **复习整理包**：
    * 数据库期末复习宝典、练习自测题与大题合集。
    * [简答题 30分.pdf](3%20数据库原理/简答题%2030分.pdf)：总结了关系代数、数据库安全等级、备份恢复机制等占期末 30 分大题的纯文字考点，备考利器！
* **📂 实验课文件夹真实资料盘点**：
  * **实验一至实验三（SQL基本操作与复杂查询）**：使用 SQL Server 2022 进行，包含建库、建表、外键级联设置；SELECT 复杂连接（INNER/LEFT JOIN）、GROUP BY 配合 HAVING 聚合过滤、嵌套子查询。
  * **实验四至实验五（完整性与安全性）**：实体完整性与参照完整性配置、自定义 CHECK 约束；使用 `GRANT` 与 `REVOKE` 授予/回收特定用户对特定表格的 SELECT/UPDATE 权限，确保安全性。
  * **实验六（数据库编程）**：T-SQL 控制结构、声明游标（Cursor）进行逐行遍历处理、以及编写带输入输出参数的**存储过程（Stored Procedure）**。
  * **实验七（综合设计 - 《音乐库管理系统》）**：大作业报告模板。详尽展示了一个**音乐库管理系统**的开发流程：
    * **需求分析**：细分为用户管理模块（登录/注册/退出）、歌曲管理模块（歌名、歌手、时长、发布期）、艺术家信息管理。
    * **逻辑设计**：规划了用户表、歌曲表、歌手表以及它们的多对多关联关系，实现了规范化，并写出建库建表的标准 SQL 实现脚本。
* **💡 备考突破秘籍**：
  1. **关系代数（必考）**：掌握关系代数的五种基本操作（并 $\cup$、差 $-$、笛卡尔积 $\times$、投影 $\pi$、选择 $\sigma$）以及连接 $\bowtie$、除 $\div$ 的推导，期末会有专门的关系代数大题。
  2. **SQL 复杂语句编写（占分超 30%）**：嵌套查询（如带 `IN`、`EXISTS` 的子查询）、关联查询时必须对具有歧义的字段加表名前缀限定、GROUP BY 字段必须与 SELECT 非聚合字段保持一致（参考 `SQL.docx` 中的案例）。
  3. **关系模式规范化理论（大题核心）**：
     * **函数依赖**：求解关系模式的属性闭包 $X^+$，寻找候选码（Candidate Key）。
     * **范式判定**：根据定义判定关系模式达到了第几范式：
       * **1NF**：属性不可再分。
       * **2NF**：消除了非主属性对码的部分函数依赖。
       * **3NF**：消除了非主属性对码的传递函数依赖。
       * **BCNF**：消除了主属性对码的部分和传递函数依赖（即所有函数依赖的决定因素都包含码）。
       * 掌握将不规范的关系模式无损连接且保持函数依赖地分解为 3NF/BCNF 的具体算法。
  4. **并发控制与封锁协议**：理解三级封锁协议的内容，以及排他锁（X锁）、共享锁（S锁）如何解决“丢失修改”、“不可重复读”和“读脏数据”这三种并发不一致性问题。

  1. **振动与机械波**：写出简谐振动的运动学方程，计算相位、速度和加速度；根据波动方程求振幅、频率、波长并画出波形图，注意判断波的传播方向与初相的对应关系。
  2. **波动光学（极重要）**：
     *   **干涉**：杨氏双缝干涉的条纹间距公式 $\Delta x = \frac{D}{d}\lambda$；薄膜干涉中的光程差公式（注意垂直入射与斜入射时半波损失 $\frac{\lambda}{2}$ 的加减）。
     *   **衍射**：单缝衍射的暗纹条件 $a \sin\theta = \pm k\lambda$，光栅衍射的公式 $d \sin\theta = \pm k\lambda$。
  3. **热学与热力学**：利用能量均分定理计算分子的平均动能与内能；利用热力学第一定律 $Q = \Delta E + W$ 计算等温、等压、等容和绝热过程的吸放热量；熟记卡诺循环效率计算公式 $\eta = 1 - \frac{T_2}{T_1}$（常考大题）。
  4. **狭义相对论**：熟练运用时间膨胀公式（钟慢效应）、长度收缩公式（尺缩效应）与质能方程 $E = mc^2$ 进行动能和静能的换算，分清“固有时间”与“固有长度”的基准。

### 4. 🟡 工程数学（一）（3 学分）
理科同学最重要的数学基石课程，包含《线性代数》与《概率论与数理统计》。
* **📂 文件夹内真实资料盘点**：
  * **备考书籍库**：包含经典的《线性代数》教材 PDF、针对课后难题的配套辅导书 PDF、以及高价值的《线性代数复习资料.pdf》和《线性代数知识卡.pdf》（汇总了行列式展开、逆矩阵公式、向量组线性相关性判定定理，适合临考前速记）。
  * **线代资料整理包**：
    * `学长线代资料` 目录：包含 01期末PPT讲义、02单元课后练习题、03历年线代期末真题、04学长自用复习材料。
* **💡 备考突破秘籍**：
  1. **行列式与逆矩阵**：掌握行列式性质（特别是转置、按行展开规则）和利用初等行变换或伴随矩阵求逆矩阵的方法，重点防范计算失误。
  2. **线性方程组的解（大题高发区）**：利用克莱姆法则或高斯消元法求带参数（如 $\lambda$, $a$）的方程组的通解。分清“无解”、“唯一解”与“有无穷多解”对应矩阵秩的条件（$r(A) = r(A|b)$ 等）。
  3. **向量空间的特征值与特征向量**：求解特征多项式 $|A - \lambda I| = 0$ 得到特征值，进而求出对应的特征向量；掌握矩阵可相似对角化的判定条件与求过渡矩阵的步骤。

## 💡 学长寄语

1. **“代码是调出来的，不是看出来的”**：第三学期的数据结构实验分量极重。千万不要只看网上的答案，直接双击打开我们的 [顺序表_源代码](1%20数据结构实验/00实验一/顺序表_源代码) 和 [二叉树_源代码](1%20数据结构实验/03实验四/二叉树_源代码)，自己在 IDE 里断点调试，只有自己彻底弄清楚双指针删除链表结点和递归算二叉树宽度的逻辑，你才能在理论考场上秒杀那些大题！
