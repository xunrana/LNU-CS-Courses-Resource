## 📚 核心专业大课真实资料盘点与备考攻略

### 1. 🟢 操作系统原理 & 实验（3 + 0.5 学分）
操作系统管理着计算机的底层资源，本课程带你彻底理解多任务并发控制、逻辑地址转换、物理页分配及中断在系统内部的衔接。
* **📂 理论课文件夹真实资料盘点**：
  * **教学课件与材料**：
    * `雨课堂资料`：涵盖第 1 章到第 8 章的精简总结 PPT，内含对 wait 和 signal 信号量同步原语的经典参考答案。
    * `操作系统1`：包含 14 级操作系统复习纲要、18 级计算考点大总结，以及第 1, 2, 3, 5, 6, 7, 8 章大段核心考点总结 PDF（均在 10MB 至 23MB 之间，覆盖进程、内存、文件、死锁和外设）。
    * [操作系统简答名解总结.docx](3%20操作系统原理/操作系统1/操作系统简答名解总结.docx)：背诵建议资料，重点梳理高级/中级/低级调度的功能，临考背记有助于应对名词解释与简答题。
  * **历年真题分类库**：
    * **徐红艳老师 15级计算机与软件 历年期末 A/B 卷真题及详细标准答案**。
    * **期末专项题目分类**：分为 `1历年试卷选择题.docx`、`2历年试卷填空题.docx`、`3历年名词解释&简答.docx`、`4历年应用题.docx`、`5历年算法题.docx` 以及雨课堂课堂测试题。考前分类刷题，极其高效！
* **📂 实验课文件夹真实资料盘点（4次C++上机实验报告与源码）**：
  * **[实验1：进程控制/调度](0.5%20操作系统原理实验/实验1)**：基于电平或状态位控制模拟进程的创建、就绪、阻塞和消亡，包含 C++ 源码及 61KB 报告。
  * **[实验2：动态分区分配算法模拟](0.5%20操作系统原理实验/实验2)**：使用 C++ 模拟实现**首次适应分配算法 (First Fit)** 和 **最佳适应分配算法 (Best Fit)** 的主存储区空间分配与回收。
  * **[实验3：虚拟存储管理 (页面置换算法)](0.5%20操作系统原理实验/实验3)**：编写算法模拟**先进先出算法 (FIFO)** 和 **最近最久未使用算法 (LRU)** 对虚拟内存页面的置换，分析在不同进程分配物理块数（如3个或4个）下的缺页率。
  * **[实验4：磁盘调度算法模拟](0.5%20操作系统原理实验/实验4)**：模拟磁盘寻道优化过程，实现**先来先服务 (FCFS)**、**最短寻道时间优先 (SSTF)** 和 **电梯扫描算法 (SCAN)**，计算磁头平均寻道长度。
* **💡 备考建议**：
  1. **进程同步与经典信号量设计（高频大题，分值通常较高）**：熟练掌握经典的生产者-消费者问题、读者-写者问题、哲学家进餐问题和理发师问题。复习时能够默画 PV 原语逻辑，理解互斥信号量（通常初值为1）与同步信号量（根据缓冲池/队列初值设定）的联调。
  2. **避免死锁与银行家算法（常考计算题）**：熟练掌握银行家算法（Banker's Algorithm）的检查机制：当进程提出资源请求时，计算其系统的安全性。考前一定要多练几次构建 $Need$ 矩阵，通过安全序列检查求出安全状态序列。
  3. **虚拟内存与页表寻址（重要计算题）**：计算多级页表下的页表项和地址转换逻辑。给定逻辑地址（十六进制/十进制）与页大小，熟练推导出页号与页内偏移量，进而结合物理块号转换出物理地址。
  4. **页面置换与缺页率计算**：根据给定的页面走向序列，手动推演并画出 FIFO 和 LRU 置换的物理块状态图，细心统计缺页中断次数和命中率（注意区分“Belady异常”现象）。

### 2. 🟢 编译原理 & 实验（3 + 0.5 学分）
编译原理是计算机系中理论性较强的必修课之一，带你探索如何将高级语言编译为目标机器码。
* **📂 理论课文件夹真实资料盘点**：
  * **教案课件库**：包含 Part1 到 Part10 完整的编译原理教学 PPT，涵盖词法分析、LL(1) 语法分析、LR(1) 算符优先、中间代码生成（四元式/三地址代码）与垃圾回收。
  * **B站复习手写板书 PDF 包**：
    * 包含 `1语法树和消除左递归回溯.pdf`、`2first集和follow集.pdf`、`3构造LL(1)分析表完整过程.pdf`、`4算符优先文法（构表+构图）.pdf`、`5构造LR(0)和SLR(1)分析表.pdf`、`6构造LR(1)和LALR(1)分析表.pdf` 以及 `10逆波兰表示法和表达式四元式.pdf`。这组资料适合系统梳理文法演变。
  * **语法与语义翻译大题详析**：
    * [分析.docx](3%20编译原理/分析.docx)：包含有向文法（如 $S \to aBc | bAB$）的**递归下降分析程序**手写代码结构，以及如何将 $S$ 属性定义的语义规则改写为 LR 翻译器下的栈操作代码。
    * [递归下降分析程序.txt](3%20编译原理/递归下降分析程序.txt)：存储了递归分析的 C/C++ 伪代码模版。
  * **考前冲刺包**：
    * 包含 `编译原理复习题(经典).doc`、`简答题.docx`、`选择题.docx` 及 `判断题.docx`。
    * **张昕老师 11 软件至 15 计算机 历年期末试卷 A/B 卷共 12 套真题与标准详析**，临考前一定要仔细刷几套真卷，题型如出一辙。
* **📂 实验课文件夹真实资料盘点（从零编写分析器）**：
  * **实验一：词法分析器 (Lexer)**。使用 C/C++ 实现输入源程序流，自动识别关键字、标识符、常数及运算符，输出二元组（类别编码，字符或数值）。
  * **实验二：LL(1) 语法分析程序**。根据给定的文法构建 FIRST 集、FOLLOW 集与 M 分析表，读取 `input.txt`，模拟控制栈的匹配过程。
  * **实验三：算符优先分析程序**。计算各非终结符的 FIRSTVT 和 LASTVT 集合，构造算符优先关系表，实现语法符号的规约判定。
  * **实验四：递归下降或 LR 语法分析程序**。设计并实现文法的递归下降子程序或构建 SLR(1) 项目集规范族与分析表。
* **💡 备考建议**：
  1. **正规式、NFA与DFA的相互转换（高频大题）**：掌握利用 Thompson 算法将正规式转化为 NFA，以及利用子集构造法将 NFA 确定化为等价的 DFA，最终使用分割法进行 DFA 最小化（经典高频大题）。
  2. **语法分析——FIRST 集与 FOLLOW 集的求法（高频大题，分值通常较高）**：
     * 精确求取每个非终结符的 $FIRST$ 集合与 $FOLLOW$ 集合；
     * 根据集合判定文法是否为 LL(1) 文法并构造 $M$ 预测分析表；
     * 掌握消除文法左递归（直接与间接左递归）和提取左公共因子的算法步骤。
  3. **LR 分析器系列分析（压轴难点）**：
     * 构建 LR(0) 项目集规范族与识别活前缀的 DFA 状态转移图；
     * 区分并理解 LR(0)、SLR(1)、LR(1) 和 LALR(1) 分析表的构造方法和冲突处理（移进/规约冲突、规约/规约冲突）。
  4. **语法制导翻译与四元式生成**：掌握中间代码表示（如三地址码、逆波兰表示法、四元式序列），多做表达式语义规则翻译的推导演练。

### 3. 🟢 最优化理论（2 学分）
主要研究如何利用数理模型和算法在各种约束条件下求解函数的极值。
* **📂 理论课文件夹真实资料盘点**：
  * **分章核心 PPT 课件**：
    * 涵盖最优化导论、凸集与凸函数（判定方法）、一维搜索方法（黄金分割法/斐波那契法）、无约束优化（最速下降梯度法、共轭方向法、Powell法、Newton法、拟牛顿DFP/BFGS法）、线性规划（单纯形法）、约束优化（罚函数法/障碍函数法）、整数规划（分支定界法）、现代优化启发式算法（遗传算法、模拟退火、蚁群算法）。
  * **作业与考核**：
    * 平时第一次、第二次作业解答 PDF（内含线性规划表格法和拟牛顿矩阵迭代推演过程）。
    * **社会现实场景“真实问题”记录表**：
      * 选题1：大规模制造业中的**库存管理与供应链优化问题**（利用整数规划与遗传算法降低运营成本）。
      * 选题2：大型城市核心路网的**交通流量管理与信号灯配时优化**（采用启发式搜索与非线性规划）。
* **💡 备考建议**：
  1. **梯度下降与牛顿法迭代（核心计算）**：掌握一元及多元函数的梯度 $\nabla f(x)$ 和 Hessian 矩阵 $\nabla^2 f(x)$ 的求解，熟练进行 Newton 法的一步迭代计算。
  2. **线性规划单纯形法（高频大题）**：理解单纯形表的建立规则、选择入基变量与出基变量的最小比值原则（$\theta$ 判定），多练几次单纯形表迭代直到所有判别数非正。

### 5. 🟢 人工智能（2 学分）
主要探索计算机如何模拟人类智能，涵盖逻辑演绎、状态搜索和机器学习分类器。
* **📂 理论课文件夹真实资料盘点**：
  * **名师期末真题库**：
    * 收录了任秀丽老师、陈廷伟老师在 13级、14级、15级 等多个班级考核下的**十余套期末 A/B 卷真题与标准答案详析**。考前精做陈廷伟老师和任秀丽老师的真卷，能更好地把握题型脉络。
* **💡 备考建议**：
  1. **一阶谓词逻辑与归结原理（高频证明大题）**：将自然语言句子转化为一阶逻辑谓词，写出子句集，利用消解（归结）原理通过反证法（添加否定目标子句）得出空子句（$\square$），注意变量替换规则。
  2. **启发式搜索与 A\* 算法（核心难点）**：掌握估价函数 $f(n) = g(n) + h(n)$ 每一个算子的含义，熟练求解八数码问题（以错位棋子数或曼哈顿距离为 $h(n)$）的搜索树演变。
  3. **机器学习决策树（计算题热点）**：熟记信息熵（Entropy）及信息增益（Information Gain）的计算公式，根据特征表节点手动构建决策树。

## 💡 学长寄语

1. **“编译原理的实验建议尽早动手”**：大三的课表非常密集，而编译实验（词法分析器、预测分析表、递归下降分析）对于代码的解耦设计有较高要求。不要等到考前最后一周才开始写。多研究我们提供的 [C++词法分析代码](0.5%20编译原理实验/实验1) 和 [递归下降程序框架](3%20编译原理/递归下降分析程序.txt)，理清控制流是如何利用子程序匹配非终结符的。
2. **“OS 的精髓在于临界资源同步”**：操作系统中经常发生多线程竞争，经典 PV 原语的编写不仅是考试难点，也是求职面试并发控制的高频问题。自己尝试把生产者-消费者与读者-写者问题用不同的信号量在纸上写一写，搞清同步信号量与互斥信号量的本质区别。
3. **“最优化和人工智能是真卷的胜利”**：最优化有很多求导和单纯形表迭代，计算时要极为细心，单纯形表的换基比值很容易算错；人工智能的考试题型高度固定，重点研究陈廷伟和任秀丽老师的真题集，做完五套真卷后，人工智能便能拿到不错的成绩。
4. **系统级能力的提升**：大三秋季这一年是大学阶段专业核心的高峰。扎实学完编译与操作系统后，你的架构设计与底层控制能力会明显提升，大三下学期及之后的毕设与项目压力也会降低。
