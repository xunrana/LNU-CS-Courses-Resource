# 贡献指南

感谢你愿意补充课程资料。为了让仓库长期可维护，请按下面规则提交。

## 提交前检查

- 确认资料放在对应学期和课程目录下。
- 文件名尽量包含课程名、资料类型、年份或说明。
- 不上传含姓名、学号、手机号、邮箱、身份证号等个人隐私的信息，必要时先脱敏。
- 不上传未获授权的付费教材、电子书、教师课件、当学期尚未完成考试的试卷。
- 新增超过 50 MB 的单文件会触发审计警告，需说明来源、授权与保留必要性；不要新增超过 100 MB 的单文件。

## 推荐目录归类

- `课件`：PPT、教学日历、教学大纲等。
- `笔记`：复习提纲、知识点总结、错题整理。
- `试卷`：历年试卷、答案、评分标准、试卷分析。
- `实验`：实验指导书、实验报告模板、实验代码。
- `大作业`：项目源码、项目报告、演示材料。

## 命名建议

- 课件 / 笔记：`课程名_类型_说明.扩展名`
- 实验：`课程名_实验序号_主题.扩展名`
- 试卷：`课程名 学年学期 年级 专业 附加信息.doc`

示例：

- `数据结构_课件_第3章树.pdf`
- `计算机网络_实验3_Wireshark抓包报告.docx`
- `数据库原理 2023-2024-1 22级 计科 试卷A.doc`

## README 更新

如果新增资料会改变课程资料状态，请同步更新对应 README 中的资料清单标记。

提交前建议运行：

```powershell
powershell -ExecutionPolicy Bypass -File scripts/audit.ps1
```
