# 📘 第六学期 · 大三春季课程与工程前沿大集成手册

> **大三第六学期（大三春季）是整个大学阶段专业工程化落地、前沿技术（人工智能/机器学习）探索的“集大成期”！**

> 在本学期中，你将面临多达 **9 门科目** 的融合性挑战。专业课的重心全面偏向“网络与分布式系统工程”以及“数据科学与智能计算”。以《计算机网络》为底层基石，结合《机器学习》《大数据导论》两大数据科学课程，辅以《Python程序设计》《拓展实验》等工程实操，逐步训练网络工具开发与智能分类模型搭建能力。本手册将带你全面盘点第六学期真实资料，拆解工程备考与项目完成建议。

## 📚 核心专业大课真实资料盘点与备考攻略

### 1. 🟢 计算机网络 & 实验（3 + 0.5 学分）
本课程探讨计算机节点间如何通过七层/五层协议栈实现跨物理介质的数据包传递、路由寻址与端到端流控。
* **📂 理论课文件夹真实资料盘点**：
  * **教学课件与材料**：
    * 官方大纲课件 PPT（第 1 章到第 3 章，覆盖物理层、链路层、网络层及传输层核心内容）。
    * 湖科大名师版分章复习背诵手册（第 1 章到第 6 章独立 Word，针对计网高阶知识点归类极好）。
  * **考前归纳资料**：
    * 计网名解释与简答题总结 PDF、选择题整理 PDF 以及大题综合训练空白/带答案版本。
    * **曲大鹏老师、刘建老师 11级至14级 历年期末真题及标准详尽答案**。
  * **计网开发大作业项目（Packet Sniffer GUI）**：
    * **项目介绍**：基于 **Python + Scapy + Tkinter** 开发的图形化网卡数据包捕获、过滤与查看工具。
    * **源码核心架构**：
      * `main.py`：程序入口，创建 Tkinter 主窗口。
      * `ui_main.py`：GUI 界面与交互逻辑，负责协议过滤、列表展示和详情查看。
      * `sniffer.py`：利用 Scapy 进行数据包捕获，并将结果交给解析逻辑处理。
      * `parser.py`：解析源 IP、目标 IP、协议类型、包长度与数据包详情。
* **📂 实验课文件夹真实资料盘点 (Cisco 拓扑拓扑网络搭建)**：
  * **系统总报告**：[计网实验.docx](0.5%20计算机网络实验/计网实验.docx)。包含了 6 个大型网络仿真的配置指令、路由表解析和网络拓扑截图：
    * **实验一：虚拟局域网 VLAN 划分**：利用 Cisco Packet Tracer 规划 8 台 PC 主机、2 台 2960 交换机，验证同一 VLAN 直接通信与不同 VLAN 广播隔离。
    * **实验二：交换机链路聚合**：实现端口通道（Port Channel）绑定以拓宽带宽。
    * **实验三：静态路由与默认路由配置**：配置多路由器跳转表。
    * **实验四：单臂路由实现跨 VLAN 通信**：利用子接口技术连接不同广播域。
    * **实验五：网络地址转换 NAT 技术配置**：将私网 IP 映射转换至公网 IP，解决公网 IP 不足问题。
    * **实验六：Web服务文档与 HTTP 协议抓包分析**：基于 HTML/CSS/JS 建立简单网页并抓包分析 HTTP 会话。
* **💡 备考建议**：
  1. **数据链路层与网络层寻址（计算题重点）**：
     * 掌握滑动窗口协议（后退 N 帧 GBN 与选择重传 SR）的窗口边界计算；
     * 熟练进行**无分类域间路由选择 CIDR 子网划分**（求子网掩码、网络前缀、地址范围）；
     * 掌握路由器的路由表匹配逻辑（最长前缀匹配原则）。
  2. **传输层可靠传输与拥塞控制（期末压轴大题，分值超15分）**：
     * 重点分析 TCP 的滑动窗口、拥塞控制机制：**慢开始 (Slow Start)**、**拥塞避免 (Congestion Avoidance)**、**快重传 (Fast Retransmit)** 和 **快恢复 (Fast Recovery)**；
     * 根据拥塞窗口大小（$cwnd$）随传输轮次的变化趋势画出拥塞演变曲线（需要熟记发生超时重传和 3 个冗余 ACK 时的不同处理逻辑，必考！）。
  3. **应用层协议工作过程**：理解 DNS 域名解析的过程（递归与迭代查询的区别）；掌握 HTTP 报文格式（请求方法、状态码含义）以及邮件发送与接收协议。

### 2. 🟢 机器学习 & 大数据导论（2 + 2 学分）
大三下最亮眼的数据科学双星，将统计数学与高级算法应用在真实海量数据集上。
* **📂 机器学习大作业与资料盘点**：
  * **名师大纲与作业**：`施瀚老师教学大纲.doc`、平时课堂随堂测试题。
  * **课程核心项目《基于机器学习的诈骗短信识别系统》**：
    * **内容**：针对 SMS Spam Collection 公开数据集，进行数据预处理（类别分布、短信长度分布可视化、高频词 TF-IDF 特征提取）。
    * **分类模型**：用 Python 实现了**多项式朴素贝叶斯 (Naive Bayes)**、**支持向量机 (SVM)** 与 **逻辑回归 (Logistic Regression)** 三种机器学习算法。
    * **评估表现**：包含混淆矩阵（Confusion Matrix）、召回率（Recall）分析、ROC 曲线与性能对比柱状图的绘制代码和文档。
* **📂 大数据导论课件与作业盘点**：
  * **名师教案课件 Ch1-Ch5**：覆盖大数据基本概念、Numpy 数据处理、数据采集（网络爬虫框架）、机器学习算法库以及经典深度神经网络。
  * **大数据导论笔记整理（大 PDF）**：约 **54MB 的系统级大数据复习手写整理笔记**，适合考前系统回顾。
  * **大数据导论课程设计**：选题为《短信垃圾分类与文本特征分析研究》，包含 2MB 规格的系统开发论文与 PPT 汇报大纲。
* **📂 大数据导论实验课（李默老师指导）**：
  * 包含从 `01` 到 `06` 完整的 6 项数据分析报告与 Python 绘图结果：
    * **实验一**：网络爬虫数据采集、清洗并存为 Excel 文件。
    * **实验二**：感知机二分类与 K-近邻（KNN）算法分类评估。
    * **实验三**：朴素贝叶斯与决策树分类实践。
    * **实验四**：支持向量机 (SVM) 核函数分类评估。
    * **实验五**：无监督学习（K-Means 聚类与系统级层次聚类）。
    * **实验六**：利用真实的 `StudentsPerformance.csv` 成绩数据集，对学生表现、备考情况与最终成绩进行多维度关联分析与可视化呈现。
* **💡 备考建议**：
  1. **统计概率与贝叶斯定理**：理解条件概率，熟练利用贝叶斯公式计算文本垃圾分类概率。
  2. **分类器优化核心公式**：
     *   **感知机与 SVM**：求解最佳划分超平面，理解拉格朗日乘子与核函数（高斯核/线性核）的选择；
     *   **K-Means 聚类**：掌握欧氏距离计算及聚类中心迭代更新算法步骤。

### 3. 🟢 拓展实验（1 学分）
这门实验课以高级启发式智能搜索算法求解经典组合优化问题为核心，极具挑战。
* **📂 文件夹内真实资料盘点**：
  * **实验大作业报告**：[大作业.docx](1%20拓展实验/大作业.docx)。课题为《旅行商问题的求解分析》，详细展示并实现了三种经典算法解决 TSP 问题：
    * **算法一：最邻近算法 (NNA)**：近似贪心思想，快速生成可行解。
    * **算法二：动态规划算法 (DP)**：自底向上进行状态缓存，精确求解小规模 TSP。
    * **算法三：遗传算法 (GA)**：设计基因编码、选择（轮盘赌）、交叉（双点交叉）与变异操作，实现大规模城市下的全局最优路线寻优，并进行路径与收敛曲线的可视化。
  * **启发式 C 语言源码库**：
    * `模拟退火算法解决0_1背包问题.c`：定义温度衰减系数，引入 Metropilis 准则跳出局部最优。
    * `蚁群算法解决背包问题.c`：初始化信息素矩阵，模拟蚂蚁寻优行为。
    * `遗传算法解决TSP问题.c`：纯 C 语言高效实现选择、杂交和突变操作。

## 💡 学长寄语

1. **“计网不仅要背，更要动手抓包”**：计算机网络考点多而杂，容易记混。建议优先阅读可运行版本 [Packet Sniffer GUI 源码](3%20计算机网络/大作业/project)，了解网卡如何捕获以太网帧、如何解析出 IP 头和 TCP 头部序列号。再配合曲大鹏老师的真题集自测，复习会更有针对性。
2. **“打通 Python 爬虫到机器学习的闭环”**：把 Python 爬虫大作业跟大数据的实验一（爬虫）、实验六（大数据关联分析）、机器学习大作业（垃圾短信分类）结合起来。当你掌握“爬取数据 -> 清洗存储 -> 模型训练 -> 结果分析”这一整套闭环后，后续做数据科学或机器学习方向项目会更顺手。
3. **“启发式算法是大型项目寻优的利器”**：拓展实验中的遗传算法（GA）、模拟退火（SA）解决 TSP 和背包问题，代码逻辑相当庞大。好好阅读我们的 [遗传算法解TSP大作业报告](1%20拓展实验/大作业.docx)，搞懂轮盘赌选择和交叉互换是如何一步步迭代逼近全局最优解的，这对于以后解决高维度工业寻优问题非常有启发！
4. **把项目经历整理好**：网络、数据和机器学习相关大作业都适合沉淀成简历项目。建议保留代码、报告、截图和复盘说明，便于后续求职、保研或考研复试展示。
